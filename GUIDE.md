# 使用說明 · 物料管理平台

這是一份**功能與操作導覽**,以使用者視角說明這個平台能做什麼、每個畫面怎麼操作。技術細節請見 [`ANSWERS.md`](./ANSWERS.md);專案架構講解請見 [`README.md`](./README.md)。

> 線上版本可直接操作:**https://material-editor-d1.zeabur.app**

---

## 整體介面

畫面分成三塊:

- **左側欄(深色,固定)** — 切換兩個工作區、顯示後端連線狀態、切換深 / 淺色主題。
- **頂部列** — 顯示目前頁面、SKU 數量與庫存總值、「重新載入」按鈕。
- **主內容區** — 依左側選的工作區,顯示「材質編輯器」或「庫存看板」。

### 左側欄

| 元素 | 說明 |
|---|---|
| **材質編輯器** | 切換到估價 / 材質編輯視角。 |
| **庫存看板 WMS** | 切換到倉儲視角;若有料件低於安全庫存,右側會出現紅色數字提示需補貨項數。 |
| **後端已連線** | 綠點閃爍表示與後端連線正常,並顯示最後同步時間。 |
| **深色 / 淺色模式** | 點一下即可切換主題(側欄恆為深色)。 |

### 頂部列

- **SKU / 金額**:目前料件種類數,以及即時的庫存總價值。
- **重新載入**:重新向後端抓取最新料件資料。

## 工作區一:材質編輯器

用來編輯零件的材質與價格,並即時看到估價變化。畫面分三區,**改一處,另外兩處立刻同步**。

### 1. 模型零件清單(左)

- 列出所有零件,每列顯示**名稱、分類、單項小計、顏色方塊**。
- 點任一列即選取該零件,右側編輯器會載入它的資料(被選中的列有橘色高亮)。

### 2. 材質與屬性編輯器(右)

選取零件後,可編輯以下屬性,每次變更都會即時反映到清單與下方 BOM:

| 欄位 | 操作 | 範圍 |
|---|---|---|
| **材質顏色(Albedo)** | 點色塊開調色盤,或直接輸入十六進位色碼 | `#RRGGBB` |
| **基礎單價(Base Price)** | 數字輸入框,可用上下鍵微調 | ≥ 0,每步 10 |
| **材質複雜度加成(Multiplier)** | 拖動滑桿 | 1.0× ~ 5.0×,每步 0.1 |

- 右上角的「**已同步**」標示代表變更已存回後端(儲存中會顯示轉圈,失敗會顯示重試按鈕)。
- 最下方「**即時小計(Line Total)**」= 基礎單價 × 加成,拖滑桿時會即時跳動。

> 💡 拖動滑桿時畫面**立即**更新、但不會每次都打後端——系統會在你停下約 0.3 秒後才把變更存回,避免拖曳造成卡頓或洪水請求。

### 3. BOM 物料清單預估(下)

- 表格列出所有零件:**名稱、色碼、單價 × 加成、小計**。
- 點任一列也可選取該零件(與左側清單連動)。
- 最下方「**專案總估價(Grand Total)**」= 所有零件小計加總,任一零件改動都會即時重算。

## 工作區二:庫存看板 WMS

以倉儲視角呈現同一份料件資料,並展示**後端即時推播**的庫存異動。

### 上方:KPI 卡片

| 卡片 | 意義 |
|---|---|
| **總料件數** | 料件種類(SKU)數量。 |
| **庫存總量** | 所有料件的庫存件數總和。 |
| **庫存總價值** | Σ(單價 × 加成 × 庫存)。 |
| **低於安全庫存** | 庫存低於安全水位、需補貨的料件數;有缺貨時卡片轉為紅色警示。 |

### 中間左:各分類庫存價值佔比

- 以橫條圖顯示各分類(結構組件、外部面板…)的庫存價值,長度代表佔比。

### 中間右:IoT 即時數據流

- 模擬感測器每 1.5 秒回報庫存異動,看板會即時刷新。
- **狀態燈**:`LIVE`(綠燈閃爍,資料流進行中)/ `RECONNECT…`(連線中斷、自動重連中)/ `PAUSED`(已暫停)。
- **暫停資料流 / 啟動資料流**按鈕:手動開關即時推播。
- **事件流(Event Log)**:列出最近的庫存異動,綠色 `+` 為入庫、紅色 `−` 為出庫,附時間戳。

> 💡 這裡的庫存變化是**後端**產生並寫入資料庫後推播的,所以重新整理頁面後庫存會保留、且多個分頁會同步更新。

### 下方:庫存明細(Inventory)

逐筆列出料件的庫存狀態:

| 欄位 | 說明 |
|---|---|
| **料件 / 儲位** | 名稱、分類與儲位代碼(如 `A-01-03`)。 |
| **庫存量** | 目前庫存;低於安全庫存時顯示紅色。 |
| **水位** | 進度條呈現庫存相對安全庫存的水位。 |
| **安全庫存** | 補貨警戒線。 |
| **庫存價值** | 該料件的庫存總值。 |
| **狀態** | 綠色「正常」或紅色「需補貨」標籤;需補貨的整列以紅底標示。 |

## 常見操作情境

- **想試三區塊即時同步** → 進「材質編輯器」,拖動某零件的 Multiplier 滑桿,觀察左側清單價格與下方 BOM 總價同步跳動。
- **想看即時數據** → 進「庫存看板 WMS」,看事件流持續滾動、KPI 與長條圖即時刷新;按「暫停資料流」可凍結觀察。
- **想驗證資料持久化** → 在編輯器改某零件的單價,重新整理頁面,改動仍在(已存回後端)。
- **想換配色** → 左下角點「深色 / 淺色模式」切換主題。
